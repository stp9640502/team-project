產業新尖兵-全端開發

Web/Mobile App 是近年來最為普遍的軟體開發議題，業界對於 App前、中、後端的開發者需求甚高。 本課程從基礎的網頁設計與 Python 程式設計開始，最後講到結合 Web 伺服器與資料庫的 3-tier Web Application 開發，並且以 Python 程式語言貫穿所有重要的課程，藉此達成具有一貫性的課程設計，透過此課程的訓練，學員可以具備進入開發 Web Application 的能力。
